# MERN Todo App with Authentication

## Description
This project is a Todo application built with the MERN stack (MongoDB, Express, React, Node.js).

It will include authentication with JWT and allow users to manage their personal tasks.

---

## Current Progress

Backend initialization completed:

- Express server created
- Environment variables configured
- MongoDB connection established
- Basic route implemented
---
## Technologies Used

- Node.js
- Express
- MongoDB
- Mongoose
- dotenv
- cors
---

## Installation

Clone the repository: git clone https://github.com/yourusername/mern-todo-auth.git

Install dependencies:
- cd backend
- npm install

Run the server: npm run dev
- Server runs on: http://localhost:5000
- Test route: GET / Hello World!
